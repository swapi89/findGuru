How to run front end code on your system :-

1) npm install
2) npm start

Pre- requisites :
1) Node is installed, if not install node and check version : node --version
2) NPM is installed, check version : npm --version
3) Install typescript : npm install -g typescript , version : tsc --version
4) install apm , if not npm install -g apm
5) install typescript in atom : apm install atom-typescript
6) install git.
